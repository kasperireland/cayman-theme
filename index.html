Note.ovh

Take notes in a hurry with Note.ovh browser extension.



For Google Chrome

Download the .crx chrome extension file by clicking the button at the top of this page or by clicking right here. Then, open a new tab and in the search bar at the top, type in (or copy paste) chrome://extensions

Make sure the developer mode checkbox is checked, and then drag the file you downloaded (note.ovh.crx) onto the screen, and you are all set! Just don't forget to click the confirmation 'add' button.
